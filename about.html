Who am I?

Professionally, I'm a software engineer. My Python expertise is at the expert level. I've been working on hardware design using the software paradigm for more than two years. I am well knowledgeable about the RISC-V Instruction Set Architecture (ISA), the only open source ISA that is transforming the semiconductor industry. Constructing Hardware in Scala Embedded Language (CHISEL), a high level hardware construction language (HCL), is something I have a lot of experience with. Using the Object Oriented Programming (OOP) and Functional Programming (FP) methodologies, I have created numerous designs on CHISEL.

Additionally, I've written low-level C-based Linux kernel device drivers. I also have experience creating kernels for Real Time Operating Systems (RTOS), such as Zephyr. I've also worked on porting applications, such as the Android Open Source Project (AOSP) Emulator, to the RISC-V ISA.

What have I done?

My final year project (FYP) was "SoC-Now," an open source, web-based RISC-V SoC generator. It is entirely built on the CHISEL HCL, which generates an entire SoC based on configurations like Core, Device(s), and Bus Interconnects before taking it to FPGA Emulation.

As a member of the software team for "The Nova Project," I have also written Linux kernel device drivers and constructed the Zephyr RTOS kernel using SoC configurations.

Additionally, I was chosen for the "Porting AOSP 12 Emuator to RV64GC" project under The Linux Foundation's Mentorship Program. where I had included RISC-V 64 bit GC support for the Android 12 emulator.
